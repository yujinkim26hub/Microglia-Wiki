# 🛑 Unresolved paper requests

When a paper-ingestion request **cannot be confidently identified**, it is recorded here instead of being guessed. See `AGENTS.md §5`.

> [!warning] Rule
> Never fabricate a citation. If author/title/year/journal cannot be matched to a real record with a DOI or PMID, log it below as **needs verification** and tell the user.

## Entry format

```
### <First author> (<Year>) — "<Title>" — needs verification
- Requested: First author / Title / Year / Journal / Section (as given)
- Date logged: YYYY-MM-DD
- Searched: <queries / databases tried>
- Why unresolved: <no match | ambiguous | multiple candidates | metadata mismatch>
- Candidates found (if any): <title — DOI/PMID — why uncertain>
- Next step: <what info would resolve it>
```
